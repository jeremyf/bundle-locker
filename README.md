# Bundle::Locker

Lock Gemfile gem declarations to specific Gemfile.lock versions

## Installation

Add this line to your application's Gemfile:

    gem 'bundle-locker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bundle-locker

## Usage

$ bundle-locker /path/to/Gemfile
